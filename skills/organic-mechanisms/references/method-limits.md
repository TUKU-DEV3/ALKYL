# When EASE Fails — Memorize Instead

Source: AceOrganicChem.com "Overtime" section (2013).

## The Principle
EASE always works in principle (electron flow never changes). But for some reaction classes it is faster/safer to memorize the outcome than to reason through it.

---

## 1. Oxidation / Reduction

Memorize reagent → outcome:

| Reaction | SM | Product | Reagent | Note |
|---|---|---|---|---|
| Oxidation | 1° alcohol | carboxylic acid | CrO₃/H₂SO₄ (Jones) | harsh |
| Oxidation | 1° alcohol | aldehyde | PCC or PDC | soft, stops at aldehyde |
| Oxidation | 2° alcohol | ketone | PCC, PDC, CrO₃ | |
| Oxidation | alkene | cis-diol | OsO₄ (cat.) + NMO | syn dihydroxylation |
| Oxidation | alkene | epoxide | mCPBA | |
| Oxidation | alkene → 2 carbonyls | ozonolysis | O₃ then DMS (aldehyde) or H₂O₂ (acid) | |
| Reduction | ketone/aldehyde | 2°/1° alcohol | LiAlH₄ | harsh, reduces all C=O |
| Reduction | ketone/aldehyde | 2°/1° alcohol | NaBH₄ | soft, C=O only |
| Reduction | ester/amide/acid | alcohol/amine | LiAlH₄ | |
| Reduction | aldehyde | 1° alcohol | DIBAL-H (−78 °C) | stops at aldehyde from ester |
| Reduction | alkyne → E-alkene | Na/NH₃ | Birch-type | trans |
| Reduction | alkyne → Z-alkene | H₂/Lindlar | | cis |
| Reduction | alkene → alkane | H₂/Pd-C | | full |
| Reduction | NO₂ → NH₂ | SnCl₂ or Fe/HCl | | |

**Mnemonic**: LEO the lion goes GER — Losing Electrons = Oxidation; Gaining Electrons = Reduction.

---

## 2. Radical Reactions

Mechanism: initiation → propagation → termination. **Not** EASE (radicals ≠ classical electrophiles).

| Reaction | Conditions | Product | Regiochemistry |
|---|---|---|---|
| Alkane halogenation | Cl₂/hν | alkyl chloride | statistical (Cl) / selective 3°>2°>1° (Br) |
| Allylic bromination | NBS | allylic bromide | allylic position |
| Benzylic bromination | NBS | benzyl bromide | benzylic position |

Key: Br radical is more selective (endothermic H-abstraction) → 3° >> 2° > 1°. Cl radical is less selective (exothermic) → near-statistical.

---

## 3. Organometallic Formation

Memorize: metal inserts into C–X bond. Not a nucleophilic mechanism.

| Reagent formed | From | Metal |
|---|---|---|
| Grignard (RMgX) | R–X + Mg, Et₂O | Mg |
| Organolithium (RLi) | R–X + 2 Li | Li |
| Organocuprate (R₂CuLi) | 2 RLi + CuX | Cu |
| Organozinc (RZnX) | R–X + Zn | Zn (Reformatsky) |

**X** = halogen, almost always Cl, Br, or I (not F).

**Use cases**:
- Grignard/RLi → 1,2 addition to C=O (hard Nu)
- R₂CuLi → 1,4 Michael addition (always, never 1,2)
- Reformatsky (RZnX) → alpha-bromo ester + C=O → beta-hydroxy ester

---

## Summary: EASE vs. Memorize

| Use EASE | Memorize |
|---|---|
| SN1/SN2/E1/E2 | RedOx reagents |
| Carbonyl additions (aldol, Claisen, Michael) | Radical halogenation selectivity |
| EAS, SNAr | Organometallic formation |
| Acyl substitution | Lindlar vs Na/NH₃ selectivity |
| Any reaction with clear Nu/E+ | Ozonolysis workup conditions |
