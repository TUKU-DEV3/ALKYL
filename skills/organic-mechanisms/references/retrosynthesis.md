# Retrosynthesis

Source: AceOrganicChem.com (2013) + Clayden.

## Core Concept
Work **backwards** from target → simpler precursors. Break strategic bonds (disconnections). Assign charges to fragments (synthons) → identify real reagents.

## Two Operations

**1. Disconnection** — break a bond, assign +/− to fragments (synthons):
- Fragment with electrons (Nu-derived) → synthon carries "−" → real molecule is nucleophile
- Fragment without electrons (E-derived) → synthon carries "+" → real molecule is electrophile
- Use resonance to validate: if synthon resembles a real reactive intermediate, you're on track

**2. FGI — Functional Group Interconversion** — change FG without breaking C skeleton:
- Alcohol → alkyl halide (SOCl₂)
- Alkene → alcohol (hydration)
- Ketone → alcohol (reduction)
- Etc. — use toolbox table

## Where to Disconnect

1. **Adjacent to functional groups** — FGs create electronegativity differences → chemistry happens there
2. **C–C bonds alpha to C=O** — aldol, Claisen, Michael disconnections
3. **C–X bonds** — Grignard, SN2 disconnections
4. Choose disconnection that gives **visibly simpler** precursor

## Rules (from book)

✓ Work backwards to an "acceptable" starting material
✓ Count carbons — don't lose track
✓ Locate FGs first — they are the best place to disconnect
✓ Place most reactive FGs in the product **last** (they won't survive earlier steps)
✓ Examine more than one pathway
✓ If stuck → FGI first, then retry

✗ Don't ignore other FGs on the molecule
✗ Don't invent chemistry "that looks like it should work"
✗ Don't use reactions that give mixtures
